# 📡 Akıllı Müşteri Yönetim ve Analiz Sistemi

Python Programlama dersi kapsamında geliştirilen bu proje, bir telekomünikasyon (Telco) şirketi için temel müşteri yönetimi ve analiz sistemi simülasyonu oluşturmayı amaçlamaktadır.

Proje içerisinde Python’un temel programlama yapıları kullanılarak:
- müşteri verilerinin yönetimi,
- fatura hesaplama işlemleri,
- churn (müşteri kaybı) analizi,
- veri organizasyonu,
- otomatik müşteri sınıflandırması
gibi işlemler gerçekleştirilmiştir.

---

# 🚀 Proje Özellikleri

✅ Müşteri bilgilerinin Dictionary yapısında saklanması  
✅ Hizmetlerin List ve Set yapıları ile yönetilmesi  
✅ VIP müşteri tespiti  
✅ Random müşteri ID oluşturma  
✅ KDV dahil otomatik fatura hesaplama  
✅ Churn (müşteri ayrılma riski) analizi  
✅ Tarih ve zaman işlemleri  
✅ Python kütüphaneleri kullanımı (`random`, `math`, `datetime`)  
✅ Döngüler ve fonksiyonlarla çoklu müşteri yönetimi
